# 世界时钟页面 - 技术设计文档

## 1. 项目概述

- **项目名称**：世界时钟页面
- **项目类型**：Web 单页应用
- **核心功能**：展示多个时区的实时时间，支持添加/删除城市时钟
- **目标用户**：需要同时查看多时区时间的用户

## 2. 技术栈

### 前端
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：CSS Modules 或 Tailwind CSS
- **时间处理**：date-fns 或 dayjs
- **部署**：静态部署

### 数据存储
- **本地存储**：localStorage 持久化用户配置

## 3. 功能设计

### 3.1 核心功能
1. **时钟卡片展示**
   - 显示城市名称、当前时间、时区偏移量
   - 实时更新时间（每秒刷新）
   - 支持多个城市同时显示

2. **城市管理**
   - 搜索并添加城市
   - 删除不需要的时钟卡片
   - 拖拽排序（可选）

3. **时间显示**
   - 12/24 小时制切换
   - 日期显示（跨日提示）

### 3.2 默认城市
- 北京 (Asia/Shanghai)
- 东京 (Asia/Tokyo)
- 伦敦 (Europe/London)
- 纽约 (America/New_York)

## 4. 界面设计

### 4.1 布局
- 顶部：标题 + 设置按钮
- 主体：时钟卡片网格布局
- 底部：添加城市按钮

### 4.2 视觉
- 卡片式设计，圆角阴影
- 渐变背景或深色主题
- 清晰的数字时钟显示

## 5. 接口设计

### 无后端接口
- 纯前端静态应用
- 时区数据来自浏览器 Intl API
