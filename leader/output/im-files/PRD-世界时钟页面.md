# 世界时钟页面 - 产品需求说明书 (V2.0)

## 1. 项目概述

- **项目名称**：世界时钟页面
- **项目编码**：PRJ-20260317100225
- **项目类型**：Web 单页面应用
- **核心功能**：以酷炫圆形钟表形式展示多个时区的实时时间，每个钟表代表对应国家/地区的独特风格
- **目标用户**：需要跨时区查看时间的用户（如远程工作者、国际业务人员、旅行者）

## 2. 功能需求

### 2.1 核心功能

| 需求编号 | 功能描述 | 优先级 |
|---------|---------|--------|
| F01 | **多时区时间展示**：同时显示至少8个时区的当前时间，以圆形钟表形式呈现 | P0 |
| F02 | **实时更新时间**：每秒自动更新显示时间，走时精确 | P0 |
| F03 | **国家风格钟表**：每个时区对应独特的钟表视觉风格（如日本樱花钟表、巴黎浪漫风格、纽约都市风格等） | P0 |
| F04 | **时区搜索添加**：支持搜索并添加新时区 | P1 |
| F05 | **时区删除**：支持移除不需要的时区 | P1 |
| F06 | **时间格式切换**：12小时制 / 24小时制切换 | P2 |
| F07 | **日期/星期显示**：显示各时区对应的日期和星期 | P2 |

### 2.2 用户交互

- 点击圆形钟表可查看该时区的详细信息弹窗
- 支持拖拽排序钟表位置
- 时区列表数据本地存储，刷新后保留
- 悬停钟表显示放大动效

### 2.3 边界情况

- 网络断开时使用本地时间继续运行
- 非法时区数据自动过滤
- 最大支持添加12个时区
- 钟表数量不足8个时，显示占位图

## 3. UI/UX 需求

### 3.1 布局设计

- **整体布局**：单页面，顶部标题区 + 圆形钟表网格
- **响应式**：
  - 桌面端：4列网格
  - 平板端：3列网格
  - 移动端：2列网格
- **钟表尺寸**：统一直径 180px（桌面）/ 140px（移动）

### 3.2 视觉风格

- **主题**：赛博朋克 + 各国文化元素融合
- **配色**：深色背景 + 霓虹光效 + 各国家主题色
- **动画**：
  - 秒钟平滑转动
  - 钟表进入时有渐显动画
  - 悬停时有脉冲发光效果
- **字体**：数字采用等宽字体，保证时间对齐

### 3.4 组件清单

| 组件 | 描述 |
|-----|------|
| 页面Header | 标题 "世界时钟" + 当前本地时间 + 设置按钮 |
| 圆形钟表 | 城市名、时区偏移、时间、日期、国家风格装饰 |
| 添加时区按钮 | 打开时区选择弹窗，样式为"+ "圆形按钮 |
| 时区选择器 | 搜索框 + 分类时区列表（按大洲分组） |
| 设置面板 | 格式切换、主题切换（明/暗） |
| 钟表详情弹窗 | 完整时区信息、相对于本地时间差 |

## 4. 技术约束

- **前端框架**：Vue 3 / React / 原生HTML/JS
- **时区处理**：moment-timezone 或 dayjs + dayjs-tz
- **动画库**：CSS Animation / GSAP
- **存储**：localStorage 持久化用户配置
- **无后端依赖**：纯前端实现
- **浏览器支持**：Chrome, Firefox, Safari, Edge 最新版

## 5. 验收标准

- [ ] 页面加载后显示默认8个时区的圆形钟表
- [ ] 每个钟表有独特的国家风格视觉元素
- [ ] 时间每秒自动更新，秒钟平滑转动
- [ ] 可成功添加新时区（支持至少20个常用时区）
- [ ] 可成功删除不需要的时区
- [ ] 刷新页面后保留用户自定义的时区列表
- [ ] 12/24小时制切换生效
- [ ] 移动端正常显示，布局自适应
- [ ] 钟表悬停有动画效果
