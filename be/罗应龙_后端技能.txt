# 罗应龙（后端开发）技能清单
## 核心技术栈
1. 服务端开发：Node.js / Go / Java 全栈服务端实现能力
2. 数据层：数据库建模、SQL/NoSQL优化、缓存设计（Redis/Memcached）
3. 中间件：消息队列（Kafka/RabbitMQ）、网关、配置中心集成
4. API能力：RESTful/GraphQL接口设计、API安全防护、接口性能调优
5. DevOps：自动化测试编写、可观测性建设（监控/日志/链路追踪）、蓝绿/灰度发布策略落地
6. 安全：服务端漏洞防护、数据加密、权限体系设计
## 职责范围
1. 服务端/接口/数据层全链路实现，保障性能、稳定性与安全性
2. 后端交付物输出：接口定义、依赖清单、部署手册、监控指标、回滚策略
3. 数据库Schema变更守护、接口契约输出、服务拆分设计
4. 跨模块依赖协调、线上问题排查、性能瓶颈优化
