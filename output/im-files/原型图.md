# 登录系统原型图

## 页面结构

```
┌─────────────────────────────────────────┐
│              登录系统                     │
├─────────────────────────────────────────┤
│  1. 登录页 (Login)                       │
│     ├── 用户名输入框                      │
│     ├── 密码输入框                        │
│     ├── 记住我 checkbox                   │
│     ├── 登录按钮                          │
│     ├── 注册链接                          │
│     └── 忘记密码链接                      │
│                                         │
│  2. 注册页 (Register)                    │
│     ├── 用户名输入框                      │
│     ├── 密码输入框                        │
│     ├── 确认密码输入框                    │
│     ├── 密码强度指示器                    │
│     ├── 注册按钮                          │
│     └── 登录链接                          │
│                                         │
│  3. 首页 (Home - 登录后)                  │
│     ├── 欢迎消息 + 用户名                  │
│     ├── 登出按钮                          │
│     └── 内容区域                          │
└─────────────────────────────────────────┘
```

## 页面流转

```
[登录页] ──(点击注册)──> [注册页]
[注册页] ──(注册成功)──> [登录页]
[登录页] ──(登录成功)──> [首页]
[首页]   ──(点击登出)──> [登录页]
```

## 组件规格

### 登录页

| 组件 | 类型 | 必填 | 校验 |
|------|------|------|------|
| 用户名 | input/text | 是 | 非空 |
| 密码 | input/password | 是 | 非空 |
| 记住我 | checkbox | 否 | - |
| 登录按钮 | button | - | 提交 |

### 注册页

| 组件 | 类型 | 必填 | 校验 |
|------|------|------|------|
| 用户名 | input/text | 是 | 4-20字符，唯一性校验 |
| 密码 | input/password | 是 | ≥8位，大小写+数字 |
| 确认密码 | input/password | 是 | 与密码一致 |
| 注册按钮 | button | - | 提交 |

## 错误提示示例

| 场景 | 提示文案 |
|------|----------|
| 用户名为空 | "请输入用户名" |
| 密码为空 | "请输入密码" |
| 用户名已存在 | "用户名已存在，请更换" |
| 密码强度不足 | "密码需≥8位，含大小写字母和数字" |
| 两次密码不一致 | "两次输入的密码不一致" |
| 登录失败 | "用户名或密码错误" |
| 账号锁定 | "账号已锁定，请15分钟后重试" |
