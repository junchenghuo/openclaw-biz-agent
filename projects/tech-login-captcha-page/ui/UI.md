# Tech Login + Captcha｜Gate-2 视觉稿/规范

版本：v0.1

目标：深色科技蓝基调 + 玻璃拟态（Glassmorphism）登录卡片；背景可承载轻量“粒子/光斑”动态；主按钮霓虹发光，强调可点击性；PC 优先并给出移动端适配规则；中文为主，预留英文/i18n 文案长度。

---

## 1. 画布与布局

### 1.1 PC（建议）
- 画布：1440×900（可自适配至 1920 宽）
- 栅格：12 列；内容最大宽 1200；左右边距 120
- 登录卡片：居中，宽 420–460（建议 440），高度随内容自适配
- 背景：全屏渐变 + 粒子点（可做轻微漂浮/视差）

### 1.2 Mobile（建议）
- 画布：375×812
- 安全边距：左右 20
- 卡片：宽 = 100%（减边距），圆角略小；按钮全宽

---

## 2. 色板（Color Tokens）

> 命名为建议；如前端已有 token 系统，可映射到现有命名。

### 2.1 基础背景
- `--bg-0`: #050B1A（页面最深底色）
- `--bg-1`: #07142E（深蓝过渡）
- `--bg-2`: #0B2A5A（远处高光蓝）

### 2.2 玻璃卡片（Glass Surface）
- `--glass-fill`: rgba(255,255,255,0.06)
- `--glass-stroke`: rgba(255,255,255,0.14)
- `--glass-highlight`: rgba(110,231,255,0.18)

### 2.3 主色与霓虹
- `--primary-500`: #22D3EE（青蓝）
- `--primary-600`: #14B8A6（青绿过渡，可用于渐变）
- `--neon-blue`: #5EE7FF
- `--neon-purple`: #7C5CFF（点缀）

### 2.4 文本
- `--text-1`: rgba(255,255,255,0.92)（主文案）
- `--text-2`: rgba(255,255,255,0.72)（次级说明）
- `--text-3`: rgba(255,255,255,0.48)（占位/弱提示）

### 2.5 语义色
- `--error-500`: #FF4D6D
- `--warning-500`: #FFB020
- `--success-500`: #2DE39B

---

## 3. 字体与排版（Typography）

### 3.1 字体建议
- 中文：PingFang SC / HarmonyOS Sans SC / 思源黑体（按系统回退）
- 英文/数字：Inter / SF Pro（按系统回退）

### 3.2 字号与行高
- H1（卡片标题）：20 / 28，字重 600
- Body：14 / 20，字重 400
- Caption（提示/辅助）：12 / 16，字重 400
- Button：14 / 20，字重 600，字距 +0.2px

---

## 4. 圆角 / 阴影 / 模糊（Radii & Effects）

### 4.1 圆角
- 页面卡片：`--radius-card`: 20（Mobile 16）
- 输入框：`--radius-input`: 14
- 按钮：`--radius-button`: 14

### 4.2 玻璃模糊
- 卡片背景模糊：16–20px（backdrop-filter: blur(18px)）
- 备注：若浏览器不支持，降级为纯半透明 + 阴影即可。

### 4.3 阴影（Shadow）
- 卡片投影：0 20 60 rgba(0,0,0,0.45)
- 细描边内阴影（可选）：inset 0 1 0 rgba(255,255,255,0.10)

### 4.4 霓虹发光（Glow）
- 主按钮外发光：0 0 24 rgba(34,211,238,0.55)
- Hover 增强：0 0 32 rgba(94,231,255,0.70)
- Focus ring：0 0 0 3 rgba(34,211,238,0.25)

---

## 5. 组件规范（Components）

### 5.1 输入框 Input
- 高度：44
- 内边距：12×14
- 默认：
  - 背景：rgba(255,255,255,0.05)
  - 描边：rgba(255,255,255,0.16)
  - 文本：`--text-1`
  - Placeholder：`--text-3`
- Hover：描边提升至 rgba(255,255,255,0.22)
- Focus：
  - 描边：rgba(34,211,238,0.65)
  - 外环：`focus ring`
- Error：
  - 描边：rgba(255,77,109,0.75)
  - 文案：`--error-500`

### 5.2 验证码区域 Captcha
- 布局：左侧“验证码输入框”（占 1fr），右侧“验证码图片/Canvas”（固定 120×44）
- Captcha 背景：rgba(255,255,255,0.04)
- Captcha 描边：rgba(255,255,255,0.14)
- 刷新：右上角小图标（12–14px），Hover 发光（rgba(34,211,238,0.5)）
- 点击提示：Caption 文案“看不清？点击更换”可选

### 5.3 主按钮 Primary Button
- 高度：44（Mobile 48）
- 填充：线性渐变（#22D3EE → #7C5CFF）
- 文本：白色 92%，字重 600
- 默认：带 `glow`
- Hover：亮度 +10%，glow 强化
- Disabled：
  - 背景：rgba(255,255,255,0.10)
  - 文本：rgba(255,255,255,0.35)
  - 无 glow

### 5.4 错误提示 Error Hint
- 位置：输入框下方 8px
- 字号：12/16
- 色：`--error-500`
- 图标：12px（可选）

### 5.5 玻璃卡片 Card（登录容器）
- 背景：`--glass-fill`
- 模糊：blur(18px)
- 描边：`--glass-stroke`
- 顶部高光：可用渐变叠层（rgba(94,231,255,0.14)）

---

## 6. 文案与 i18n 预留规则
- 标题（CN）：`欢迎登录`；EN 示例：`Welcome back`
- 输入框 label：`账号 / Username`、`密码 / Password`、`验证码 / Captcha`
- 预留宽度：英文长度按中文 1.6× 预估（避免按钮/提示换行）
- 错误提示：不超过两行；超出用省略或折叠“展开详情”。

---

## 7. 资源与切图建议

### 7.1 必须切图
- 不建议把整张背景切成位图；推荐前端用 CSS 渐变 + 少量粒子 Canvas。

### 7.2 可选资源
- `bg-orb-1.svg` / `bg-orb-2.svg`：背景光斑（SVG 可缩放）
- `icon-refresh.svg`：验证码刷新
- Logo：仅占位（由品牌方提供）

命名规范：`login__<component>__<state>@2x.(png|svg)`

---

## 8. 交互/动效建议（轻量）
- 背景粒子：低密度（20–40 个点），缓慢漂浮；鼠标轻微视差（可选）
- 卡片出现：opacity 0→1 + translateY 8px→0（200–260ms ease-out）
- 按钮：hover 时 glow 强化 + 渐变位移（200ms）
