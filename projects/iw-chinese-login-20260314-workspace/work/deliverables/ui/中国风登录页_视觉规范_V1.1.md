# 中国风登录页视觉规范 V1.1

- 项目：无限世界-中国风登录页
- 版本：V1.1（最终口径）
- 日期：2026-03-14
- 负责人：铭烟薇（UI）

## 1. 最终口径对齐
1) 账号校验优先：手机号优先  
2) 记住我：是（默认不勾选）  
3) 登录失败提示：统一模糊提示

## 2. 视觉方向（新中式克制风）
- 风格关键词：留白、宣纸肌理、朱砂点缀、青金主色、低饱和金线纹样。
- 情绪目标：稳重、可信、东方审美、科技感轻融。

## 3. 色彩系统
- 主色（Primary）：#1F3A5F（黛青）
- 强调色（Accent）：#C89B3C（鎏金）
- 背景色（Bg）：#F7F3EA（宣纸白）
- 文本主色（Text-1）：#1E1E1E
- 文本次色（Text-2）：#666666
- 错误色（Error）：#B42318
- 成功色（Success）：#067647

## 4. 字体与排版
- 中文首选：思源宋体（标题）+ 思源黑体（正文）
- 英文/数字：Inter
- 层级：
  - H1：36/44 Semibold
  - H2：28/36 Semibold
  - Body：16/24 Regular
  - Caption：14/20 Regular

## 5. 页面结构
- PC（1440）：左视觉右表单（6:4）
- Mobile（390）：顶部品牌区 + 居中表单卡片
- 登录卡片：圆角 16，阴影 0 8 24 rgba(31,58,95,0.10)

## 6. 组件态（关键）
### 6.1 输入框（手机号优先）
- 默认：边框 #D0D5DD
- Hover：边框 #98A2B3
- Focus：边框 #1F3A5F + 外环 2px rgba(31,58,95,.18)
- Error：边框 #B42318 + 错误文案

### 6.2 记住我（默认不勾选）
- Checkbox 默认未选中；点击后选中态显示金色勾选。

### 6.3 登录失败提示（统一模糊提示）
- 文案统一：`账号或密码错误，请重试`
- 不暴露“账号不存在/密码错误”细分信息。

### 6.4 主按钮
- 默认：#1F3A5F，文字白
- Hover：#274A77
- Active：#16304F
- Disabled：#98A2B3

## 7. 纹样与装饰规范
- 背景使用 4% 透明度回纹/云纹，不干扰可读性。
- 品牌印章元素仅用于角标，不进入交互区域。

## 8. 适配与可用性
- 响应断点：390 / 768 / 1280+
- 触控目标最小 44x44
- 对比度满足 WCAG AA（正文 4.5:1）
