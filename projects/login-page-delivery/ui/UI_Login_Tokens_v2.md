# Login Page UI Tokens（v2）

> 适用：登录页（Normal / Error / Loading）示意图。

## 颜色 Color

| Token | Hex | 用途 |
|---|---:|---|
| `--color-primary-500` | `#3B82F6` | 主按钮、链接、聚焦描边 |
| `--color-primary-600` | `#2563EB` | hover/active（建议） |
| `--color-bg-900` | `#0B1220` | 页面背景（上） |
| `--color-bg-850` | `#111B2E` | 页面背景（下） |
| `--color-surface-800` | `#0F172A` | 卡片背景 |
| `--color-border-700` | `#1E293B` | 卡片边框 |
| `--color-input-bg` | `#0B1324` | 输入框背景 |
| `--color-input-border` | `#22304B` | 输入框边框 |
| `--color-text-100` | `#E5E7EB` | 主文字 |
| `--color-text-400` | `#94A3B8` | 次级文字 |
| `--color-danger-500` | `#EF4444` | 错误态提示/描边 |
| `--color-success-500` | `#22C55E` | 勾选/成功提示（可选） |

## 字体 Typography

| Token | 值 | 用途 |
|---|---|---|
| `--font-family-sans` | `system-ui, -apple-system, Segoe UI, Roboto` | 全局 |
| `--font-size-title` | `28px` | 卡片标题（Welcome back） |
| `--font-size-h1` | `34px` | 页面品牌标题（Acme Console） |
| `--font-size-body` | `16px` | 输入内容/按钮文字 |
| `--font-size-caption` | `14px` | label、提示、辅助信息 |
| `--line-height-tight` | `1.2` | 标题 |
| `--line-height-base` | `1.5` | 正文 |

## 间距 Spacing

| Token | 值 | 用途 |
|---|---:|---|
| `--space-4` | `16px` | 小间距 |
| `--space-5` | `20px` | 控件内边距 |
| `--space-6` | `24px` | 模块间距 |
| `--space-8` | `32px` | 大间距 |
| `--space-11` | `44px` | 卡片内边距（建议） |

## 圆角 Radius

| Token | 值 | 用途 |
|---|---:|---|
| `--radius-sm` | `4px` | checkbox |
| `--radius-md` | `14px` | input |
| `--radius-lg` | `16px` | button |
| `--radius-xl` | `22px` | card |

## 组件尺寸 Component

| 组件 | 规格 |
|---|---|
| Input | 高 `54px`，左右 padding `18px`，边框 1px（focus/error 2px） |
| Primary Button | 高 `56px`，字重 normal，背景 primary |
| Card | 宽 `520px`，内边距 `44px` |
