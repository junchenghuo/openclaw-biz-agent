# 雅江 HR Vue3 升级冒烟清单（上线前/提测后）

> 目标：10~20 分钟内确认“系统可用、核心链路通、日志可追”

## A. 环境与版本确认
- [ ] 环境可访问，网关/域名解析正常
- [ ] 构建版本与发布单一致
- [ ] 基础依赖服务健康（认证、配置、数据库）

## B. 核心路径冒烟
- [ ] 登录成功并跳转首页
- [ ] 首页菜单可正常进入关键模块
- [ ] 员工列表可查询并打开详情
- [ ] 审批列表可加载并完成一次审批动作
- [ ] 入转调离任一主流程可成功提交并可查询状态

## C. 基础稳定性
- [ ] 页面无白屏/卡死
- [ ] 浏览器控制台无严重报错
- [ ] 关键接口无 5xx 激增

## D. 日志与观测
- [ ] 每条关键请求有 traceId
- [ ] 错误日志含错误码与上下文字段
- [ ] 前端错误/后端错误在观测平台可检索

## E. 结论模板
- 结果：`PASS / FAIL`
- 失败项：`<模块-步骤-现象-影响等级>`
- 建议动作：`回滚 / 热修 / 限流 / 继续观察`
