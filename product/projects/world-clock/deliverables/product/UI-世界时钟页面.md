## 组件详情

### 1. Header 区域
- 左侧：图标 🕐 + 标题 "世界时钟"
- 右侧：设置图标 ⚙️ + 添加图标 ➕
- 高度：60px
- 背景：主题色（浅色主题 #fff，暗色主题 #1a1a1a）

### 2. 时区卡片 (TimeZone Card)
- 尺寸：宽度自适应（网格布局），高度 160px
- 圆角：16px
- 阴影：0 2px 8px rgba(0,0,0,0.1)
- 内容布局：
  ```
  ┌────────────────────┐
  │  城市名    UTC+8  │  ← 城市名 + 时区偏移
  │  ───────────────  │
  │                    │
  │     14:30:25      │  ← 大字体时间（秒可选）
  │                    │
  │     3月 16日      │  ← 日期
  │     星期一        │  ← 星期（可选）
  └────────────────────┘
  ```
- 交互：
  - 悬停：轻微上浮 + 阴影加深
  - 点击删除按钮：显示确认气泡

### 3. 添加时区弹窗
- 居中弹出，半透明遮罩
- 内容：
  ```
  ┌─────────────────────────┐
  │  添加时区        [✕]   │
  │  ─────────────────────  │
  │  [🔍 搜索时区...]       │
  │  ─────────────────────  │
  │  ○ 北京 (Asia/Shanghai) │
  │  ○ 东京 (Asia/Tokyo)    │
  │  ○ 纽约 (America/NewYork)│
  │  ○ 伦敦 (Europe/London) │
  │  ...                    │
  │  ─────────────────────  │
  │        [ 添加 ]         │
  └─────────────────────────┘
  ```

### 4. 设置面板
- 右侧滑出或底部弹出
- 选项：
  - 时间格式：○ 12小时制 ○ 24小时制
  - 显示秒数：○ 是 ○ 否
  - 主题：○ 浅色 ○ 深色 ○ 跟随系统

## 色彩方案

### 浅色主题
- 背景：#f5f7fa
- 卡片背景：#ffffff
- 文字主色：#1a1a1a
- 文字次色：#666666
- 主题强调：#3b82f6（蓝色）
- 删除按钮：#ef4444（红色）

### 暗色主题
- 背景：#0f172a
- 卡片背景：#1e293b
- 文字主色：#f1f5f9
- 文字次色：#94a3b8
- 主题强调：#60a5fa
- 删除按钮：#f87171

## 字号规范

- 页面标题：24px, font-weight: 600
- 城市名称：18px, font-weight: 500
- 时区偏移：12px, font-weight: 400
- 时间显示：48px, font-weight: 700（等宽字体）
- 日期/星期：14px, font-weight: 400

## 响应式断点

- 桌面端：≥1024px，4列网格
- 平板端：768px-1023px，3列网格
- 移动端：<768px，1-2列网格

## 动效设计

- 卡片悬停：transform: translateY(-4px), 0.2s ease
- 时间更新：数字渐变过渡
- 弹窗出现：fade in + scale, 0.2s ease-out
- 删除卡片：fade out + collapse, 0.3s
