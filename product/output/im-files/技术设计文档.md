# 世界时钟页面 - 技术设计文档

## 1. 项目概述

- **项目名称**：世界时钟页面
- **项目类型**：单页面 Web 应用
- **核心功能**：展示多个国家的实时时钟，采用圆形钟表设计，每个钟表呈现对应国家风格
- **目标用户**：需要查看多国时间的用户

## 2. 验收标准

- 至少8国时区可用（中国、美国、英国、日本、德国、法国、澳大利亚、俄罗斯）
- 实时更新时间，每秒刷新
- 圆形钟表设计，酷炫视觉效果
- 每个钟表体现对应国家特色风格

## 3. 技术栈选择

### 前端
- **框架**：React 18 + TypeScript
- **样式**：Tailwind CSS + 自定义 CSS 动画
- **时钟组件**：自定义 Canvas 或 SVG 绘制
- **构建工具**：Vite

### 后端
- **运行时**：Node.js
- **框架**：Express.js
- **接口**：时区数据查询（可选，后端可提供时区列表）

## 5. 接口设计

### 5.1 后端 API（可选，本地时间计算也可）

#### GET /api/timezones
获取支持的时区列表

**响应**：
```json
{
  "code": 0,
  "data": [
    { "id": "cn", "name": "中国", "zone": "Asia/Shanghai" },
    { "id": "us", "name": "美国", "zone": "America/New_York" }
  ]
}
```

#### GET /api/time/:zone
获取特定时区当前时间

**响应**：
```json
{
  "code": 0,
  "data": {
    "zone": "Asia/Shanghai",
    "time": "2024-01-15T10:30:00+08:00",
    "offset": 8
  }
}
```

### 6.2 时间更新逻辑
- 使用 `setInterval` 每秒更新
- 使用 `Intl.DateTimeFormat` 获取各时区时间
- 实时计算时针、分针、秒针角度
