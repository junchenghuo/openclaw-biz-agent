# 触发关键词
- ROI 复盘
- 优先级排序
- 版本规划
- 价值评估

# 标准流程
1. 收集需求价值、影响用户规模、实施成本与风险。
2. 构建评分矩阵并给出排序结果。
3. 生成迭代建议（Must/Should/Could/Won’t）。
4. 输出可评审版本规划与风险备注。
